# Map Generation using Perlin noise + controls to move around it
This is just a sample test using p5.js

## Controls
### Keyboard
**z**: move forward

**s**: move backward

**q**: move left

**d**: move right

**a**: move up

**e**: move down

### Mouse
**middle click**: rotate

**mousewheel**: zoom in/out
