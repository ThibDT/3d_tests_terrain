# Map Generator using perlin noise
Base code is from https://github.com/CodingTrain/Rainbow-Code/tree/master/CodingChallenges/CC_11_PerlinNoiseTerrain_p5.js

## Controls
### Keyboard
z: move forward
s: move backward
q: move left
d: move right
a: move up
e: move down

### Mouse
middle click: rotate
mousewheel: zoom in/out
